# 餐厅系统

帮朋友做的餐厅系统，主要是为了满足用户在餐桌上扫码点菜的需求。

## 系统简介

餐厅系统分为前台和后台两个部分：

- 前台是用户点菜界面，无需登录即可点菜，但必须在url里带上“店铺id”和“桌号id”，不然会报404错误。
- 后台是店家管理界面，必须登录才能操作，主要有 店铺管理、用户管理、菜品管理、菜品分类、订单管理、生成二维码 等6大模块。
